近期有名字包含“美学”的某插件**疑似**将本插件的功能直接移植到ta的插件中并闭源销售，此举是对本人辛勤劳作的严重践踏，作者在此提倡大家支持并使用该功能的最初开发者的插件，即本插件。

# 用法1
1. 光标放到你要插入代码块的地方
2. 斜杠菜单输入`autocodeblock`或者`acb`，回车
3. 输入或者粘贴代码块
4. 点击确定

# 用法2
1. 在别的地方复制代码
2. 到思源粘贴，自动设置语言和高亮

# 用法3
1. 复制一段代码片段。
2. 直接粘贴到**已有代码块**内。
3. 插件会自动识别语言并更新该代码块的高亮设置。
   - *注意：此功能可以在设置中通过“代码块内粘贴自动识别”开关进行开启或关闭。*

# FAQ
Q: 粘贴后不是代码的内容被识别为代码？
A: 可以尝试思源的粘贴为纯文本快捷键（默认是Ctrl+Shift+V）,或右键-粘贴为纯文本，或者关闭自动粘贴，只使用斜杠菜单
