# 用法1
1. 光标放到你要插入代码块的地方
2. 斜杠菜单输入`autocodeblock`或者`acb`，回车
3. 输入或者粘贴代码块
4. 点击确定

# 用法2
1. 在别的地方复制代码
2. 到思源粘贴，自动设置语言和高亮
